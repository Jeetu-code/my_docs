Section 1: Thinking Style (Core Split)

  1. I enjoy solving logical problems more than expressing ideas creatively.
  2. I like understanding “how things work behind the scenes.”
  3. I prefer structured tasks over open-ended creative work.

Section 2: Creativity vs Logic Split

  1. I enjoy storytelling, visuals, or expressing ideas creatively.
  2. I would rather create content than solve technical problems.
  3. I like making things that entertain or engage people directly.

Section 3: Coding Inclination

  1. I am interested in learning programming seriously.
  2. I enjoy building things step-by-step using logic.
  3. I don’t mind sitting for hours fixing technical issues.

Section 4: Game-Specific Interest

  1. I often think about how games work internally.
  2. I am interested in game mechanics, levels, or character systems.
  3. I like the idea of creating interactive worlds.

Section 5: Communication & Visibility

  1. I enjoy speaking, presenting, or being on camera.
  2. I like sharing ideas online (videos, posts, blogs).
  3. I prefer interacting with people rather than working silently.

Section 6: Practical vs Research

  1. I prefer building useful apps/websites over doing research.
  2. I like seeing quick, real-world results from my work.
  3. I am less interested in heavy math or theory.

FINAL DECISION RULE

Pick ONLY ONE final career.

Step 1: Primary Selection

  • Highest score = your career path

Step 2: If there is a tie

Case A: Content Creator vs Any Coding Field (Web / Game / AI)

Ask these 3 tie-breaker questions (rate 1–5):

  1. I enjoy being visible (camera, social media, public presence).
  2. I prefer expressing ideas over building technical systems.
  3. I like interacting with people more than working alone.

Scoring:

  • Content Score = Q1 + Q2 + Q3
  • Coding Score = 15 - Content Score

Decision:

  • If Content > Coding → Content Creator
  • Else → Go to Coding Tie-Breaker (Case B)

Case B: Tie within Coding (Web vs Game vs AI)

Ask these 4 decisive questions (rate 1–5):

  1. I prefer building practical apps/websites used in real life.
  2. I enjoy creating games, simulations, or interactive worlds.
  3. I like deep logic, math, and working with data/models.
  4. I want faster job/freelance opportunities early.

Coding Path Scoring

  • ** Web Dev Score** = Q1 + Q4
  • ** Game Dev Score** = Q2
  • ** AI Engineer Score** = Q3

Final Coding Decision

  • Highest score wins

If still tie:

Web vs Game:

  • If Q4 ≥ 4 → Web Dev
  • Else → Game Dev

Web vs AI:

  • If Q3 ≥ 4 → AI Engineer
  • Else → Web Dev

Game vs AI:

  • If Q3 ≥ 4 → AI Engineer
  • Else → Game Dev
