# Coding Setup

## Operating System 
Why choose PoP!_OS insted of Windows.
Think of Pop!_OS as a specialized toolbox designed for creators, 
while Windows is more like a general-purpose home office.

* Ready to Work: It comes with the "brain" of your graphics card 
  (the hardware that does the AI math) already set up and ready to go,
   whereas Windows often requires a frustrating manual setup.

* More Breathing Room: It uses much less of your computer's power to run itself,
  leaving more room for your AI projects to run faster and handle bigger tasks.

* Professional Standard: Since almost all AI in the real world runs on systems like this,
  using it now means you are learning the "language" of the industry from day one.

To install POP!_OS and learn more about it follow this [link](https://pop-os.github.io/docs/getting-started/create-bootable-media/create-bootable-usb.html).
## Other Tools

Think of this combination (Alacritty, Zellij, and Helix) as a custom-built,
high-performance cockpit for your computer. Instead of using a heavy, slow "private jet" 
(like VS Code), you are using a "supersonic scout plane" that is incredibly fast and stays
out of your way.

* [Alacritty](https://github.com/alacritty/alacritty#configuration) (The Window): This is the fastest way to talk to your computer.
  It uses your graphics card to make text appear instantly, so there’s zero "lag"
  or delay when you type.
  To install it just run this command in your default terminal:

  ```
  sudo add-apt-repository ppa:aslatter/ppa
  ```
  ```
  sudo apt update
  ```
  ```
  sudo apt install alacritty
  ```


* [Zellij](https://zellij.dev/documentation/) (The Organizer): This lets you split your screen into multiple "rooms."
  You can have your code in one room, your AI training progress in another, and your
  files in a third—all in one window without getting lost.

* [Helix](https://docs.helix-editor.com/package-managers.html) (The Pencil): This is your code editor. It’s built to be used entirely with your keyboard
  (no mouse needed), which makes you much faster once you learn the "finger dances" for common tasks.
